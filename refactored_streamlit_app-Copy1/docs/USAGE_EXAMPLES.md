# Usage Examples

## Start the app

```bash
streamlit run refactored_streamlit_app/streamlit_app.py
```

## What to configure

1. Set clinker mass range and clinker phase ranges.
2. Set SCM ranges.
3. Choose curing time.
4. Adjust CO2 and cost factors if needed.
5. Select one or more optimization objectives.
6. Run the optimizer and inspect the Pareto dashboard.

## Outputs

- Pareto frontier plots
- Population evolution animation
- Parallel coordinates view
- Pareto results table and CSV export
- Benchmark comparison
- PDF technical report
